## BalletFlow
Ballet Exercises is a multi-page web app that provides detailed ballet exercise guides. Users can browse exercises by category and difficulty, view step-by-step illustrations and tutorial videos, and save their favorite exercises for quick access. The project is built with Vite, JavaScript and MongoDB for storing exercise data.

Live link: https://ehb-mct.github.io/web2-course-project-front-end-meredithlauwerens-1/homepage/index.html
